# Northwind Data Orchestrator

End-to-end ETL pipeline built on the classic Northwind dataset.
Transforms transactional data into a dimensional data warehouse
using a star schema design.

## Project Overview
- Source: Northwind transactional database
- Output: Dimensional model (star schema)
- Purpose: Analytics, BI, and ML-ready data

## Architecture
- Dimensions:
  - Customers
  - Products
  - Categories
  - Suppliers
  - Time
- Fact table:
  - Sales

## Orchestration
The ETL pipeline is orchestrated in Python with a controlled
execution order and reproducible transformations.

## Technologies
- Python
- SQL
- CSV-based warehouse
- Dimensional modeling

## Use Cases
- Business Intelligence dashboards
- Feature engineering for ML
- Analytical reporting
